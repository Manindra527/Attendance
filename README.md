# Attendance (MongoDB Atlas)

This project now stores users and attendance records online using MongoDB Atlas.

## Setup

1. Install dependencies:
   - `npm install`
2. Create `.env` from `.env.example` and set your Atlas connection string:
   - `MONGODB_URI=...`
   - Optional: `PORT=4000`
3. Start server:
   - `npm start`
4. Open:
   - `http://localhost:4000/`

The default page is `Attendance -Online-Master.html`.
